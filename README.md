# EdTech Course Engagement & Dropout Analysis Dashboard

## Project Overview
This project focuses on analyzing student engagement and dropout behavior in an EdTech platform using Power BI.

The goal is to identify **key factors influencing student dropout** and provide actionable insights to improve course completion rates.

---

## Key Insights
- Dropout Rate: **50.97%**
- Completion Rate: **49.03%**
- Higher inactivity leads to increased dropout
- Higher video completion improves course completion
- Dropout trends are consistent across courses and categories

---

## Analysis Covered
- Course-wise Dropout Analysis
- Category-wise Performance Analysis
- User Behavior Analysis (Login Frequency, Inactivity)
- Engagement Metrics (Video Completion, App Usage, Satisfaction)
- Comparative Analysis (Completed vs Dropout Users)

---

## Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization
